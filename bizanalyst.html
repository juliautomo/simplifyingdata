FAQ

Common questions

What format does my Excel file need to be in?
Any standard .xlsx file works. We parse it as-is — no special column names or formatting required. Sales data, inventory, returns — upload whatever you have.
Do I need an Anthropic API key?
Only for the AI chat feature. The dashboard, KPIs, and charts all work without it. If you want AI answers, you'll need a free Anthropic account — usage costs roughly $2/month billed directly by Anthropic.
How does the free trial work?
You get 7 days free with full access — no credit card required upfront. After the trial, it's IDR 500,000/month. Cancel anytime before the trial ends and you won't be charged.
Who can see my data?
Only you. Each account has its own private storage. Your data is never shared, pooled, or used for training.
Can I try it before buying?
Yes — there's a live demo you can explore with sample data before purchasing.
